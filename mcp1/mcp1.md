### 方案一：【AI 播客生成器】(基于 edge-tts 和 pydub)

**目标：** 将对话文本转为双人对话的 MP3 音频。

**复制以下 Prompt 给 AI：**

> **角色设定**：你是一个 Python 后端专家，擅长多媒体处理。
>
> **任务**：请使用 Flask 框架编写一个 API 接口，实现“AI 播客生成”功能。
>
> **技术栈要求**：
> 1.  Web 框架：`Flask`
> 2.  TTS 引擎：`edge-tts` (必须使用这个库，因为它是免费且高质量的)
> 3.  音频处理：`pydub`
> 4.  异步处理：`asyncio`
>
> **接口定义**：
> - 路由：`POST /api/podcast`
> - 入参 JSON 格式：
>   ```json
>   {
>     "script": [
>       {"role": "male", "text": "你好，今天天气怎么样？"},
>       {"role": "female", "text": "天气非常好，适合出去玩！"}
>     ],
>     "bgm_volume": -15  // 背景音乐音量，默认-15dB
>   }
>   ```
>
> **核心逻辑（一步步实现）**：
> 1.  创建一个 Flask 应用。
> 2.  定义一个异步函数来生成语音：
>     - 遍历 `script` 列表。
>     - 如果 `role` 是 "male"，使用 `edge-tts` 的中文男声（推荐 `zh-CN-YunxiNeural`）。
>     - 如果 `role` 是 "female"，使用 `edge-tts` 的中文女声（推荐 `zh-CN-XiaoxiaoNeural`）。
>     - 将每段文字生成的音频保存为临时 mp3 文件。
> 3.  使用 `pydub` 进行音频拼接：
>     - 按顺序把生成的临时音频片段拼接到一起。
>     - 在每段对话之间增加 500ms 的静音间隔。
> 4.  (可选) 如果本地存在一个名为 `background_music.mp3` 的文件，将其作为背景音乐循环铺在对话底下，并调整背景音乐音量以免盖过人声。
> 5.  将最终合成的音频导出为 `final_podcast.mp3`。
> 6.  接口最终返回该文件的下载流（使用 `send_file`）。
>
> **注意事项**：
> - 请处理好临时文件的清理。
> - 请提供 `requirements.txt` 内容。
> - 代码中需要有详细的中文注释。
